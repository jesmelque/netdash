Welcome to NetDash!

You can use this as a starting point to create your NetDash module. Explore its structure and read its comments to understand the NetDash Module conventions.

To learn more, clone the NetDash Example modules: 

cd netdash
git clone git@github.com:netdash/netdash-examples.git
mv netdash-examples/* .
This will place all of the example modules in your Django apps directory. Your directories should look like this: 
.
├── docs
├── netdash
│   ├── example_devices_dummy
│   ├── example_devices_netbox_api
│   ├── example_devices_snmp
│   ├── example_error_import
│   ├── example_error_no_app_name
│   ├── example_hostlookup_dummy
│   ├── hostlookup_abstract
│   ├── hostlookup_netdisco
│   ├── my_custom_nd_module
│   ├── netdash
│   ├── netdash-examples
│   ├── netdash_api
│   └── netdash_ui
└── netdash_module_template
    ├── api
    ├── migrations
    └── templates
Example modules can then be enabled in NETDASH_MODULES in settings.py: 

NETDASH_MODULES = [
    'example_devices_dummy',
    'example_hostlookup_dummy',
]
